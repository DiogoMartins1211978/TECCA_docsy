O Goldydocs disponibiliza uma interface web única que oferece visibilidade sobre métricas de temperatura das papas, tamanho das cadeiras e suavidade das camas! Podes até descobrir quem tem andado a comer a **tua** papa.

(Infelizmente, o Goldydocs não é um projeto real, mas podes usar este site como exemplo para criares os teus próprios sites reais com o [Docsy](https://docsy.dev))
